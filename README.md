## What these scripts do
  - Six read-only SQL scripts that collect diagnostic data from your DB2 instance.
  - No DDL, No DML, No Schema changes.
  - Safe to run on production.

## What I audit
  - Instance & version health.
  - Bufferpool hit ratios & memory configuration.
  - Top SQL by CPU, elapsed time, and rows read.
  - Tablespace utilization over 90%.
  - HADR configuration and log gap.
  - Security privileges and audit policies.
  - Backup history and recovery readiness.
